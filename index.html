✨ Key Features

Universal Platform Support

  • Mobile & TV Ready: A single APK that adapts its UI perfectly whether you are using a touchscreen phone or a D-Pad remote.
  • Premium UI/UX: Built with Jetpack Compose, featuring beautiful glassmorphism effects, smooth animations, and a sleek dark theme.

Advanced Live TV Engine

  • Robust Playlist Parsing: Supports M3U, PLS, and JSON formats.
  • DRM Support: Widevine, ClearKey, and PlayReady encrypted channels.
  • Dynamic Stream Bypasses: Automatically resolves complex tokens under the hood.

Movies & TV Series (VOD)

  • TMDB Integration: Rich metadata including backdrops, cast, genres, and organized seasons/episodes.
  • Dynamic Addon System: Remote updates via external JavaScript engine (QuickJS) without needing an app update!

Powerful Video Player (ExoPlayer)

  • Picture-in-Picture (PiP): Keep watching while using other apps (Mobile).
  • Aspect Ratio Control: Toggle Fit, Stretch, and Zoom modes easily.
  • Track Selection: Switch audio tracks & video qualities on the fly.
  • Custom TV Controls: Auto-hiding, remote-friendly interface.

In-App Auto Updates

No need to check for updates manually! The app will notify you when a new release is available on GitHub and allow you to download and install it directly from within the app. You can also manually check for updates from the Side Drawer.

⚙️ Installation Guide

  1. Download the streamBD-iptv-v1.apk file from the download button above.
  2. Transfer it to your Android TV or open it directly on your Mobile device.
  3. If prompted, allow installation from "Unknown Sources" in your settings.
  4. Launch the app and enjoy premium streaming! 🎉
